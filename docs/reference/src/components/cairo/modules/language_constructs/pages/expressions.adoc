= Expressions
 
Expressions produce values. They are used in variable initializers, function arguments, control
flow constructs, and other contexts where evaluation yields a result.
 
This overview mirrors the language syntax as specified in
`crates/cairo-lang-syntax-codegen/src/cairo_spec.rs` (the `Expr` enum and its variants), and links
to detailed subpages where available.
 
== Categories of expressions
 
- **Literals**. Immediate values such as numbers, booleans and short strings. See
  xref:pages/literal-expressions.adoc[Literal expressions].
- **Path expressions**. Names and qualified paths that resolve to values or constants. See
  xref:pages/path-expressions.adoc[Path expressions].
- **Parenthesized expressions**. Override the default precedence and grouping. See
  xref:pages/parentheses.adoc[Parenthesized expressions].
- **Operator expressions**.
  - Unary operators include logical not, bitwise not, negation, reference, address-of, and related
    forms as defined by the syntax. See xref:pages/operator-expressions.adoc[Operator expressions] (ðŸš§)
    and xref:pages/negation-operators.adoc[Negation operators].
  - Binary operators include arithmetic, comparison, logical and bitwise families. See
    xref:pages/operator-precedence.adoc[Operator precedence] and related operator pages (ðŸš§).
  - Error propagation uses the postfix `?` operator. See xref:pages/error-propagation-operator.adoc[Error propagation] (ðŸš§).
- **Function and method calls**. Invocation with positional or named arguments, including references.
  See xref:pages/function-calls.adoc[Function calls] and xref:pages/method-calls.adoc[Method calls].
- **Member access and indexing**. Field selection with `.` and indexing with `[]` where supported.
  See xref:pages/member-access-expressions.adoc[Member access expressions].
- **Tuple and struct construction**. Tuple literals and struct constructor calls. See
  xref:pages/tuple-expressions.adoc[Tuple expressions] (ðŸš§) and
  xref:pages/struct-expressions.adoc[Struct expressions] (ðŸš§).
- **Array expressions**. Fixed-size array literals. See xref:pages/array-expressions.adoc[Array expressions].
- **Blocks**. Braced sequences of statements with an optional tail expression. See
  xref:pages/block-expression.adoc[Block expression].
- **Control flow expressions**. `if`/`match` and loop forms (`for`, `loop`, `while`) evaluate to
  values. See xref:pages/if-expressions.adoc[If expressions] (ðŸš§),
  xref:pages/match-expressions.adoc[Match expressions] (ðŸš§),
  xref:pages/for-loop-expressions.adoc[For loop expressions] (ðŸš§).
- **Closures**. Anonymous functions with parameter lists and an optional return type.
